# Schedule Wizard

## ¿Qué es Schedule Wizard?

**Schedule Wizard** es una herramienta pensada para estudiantes que quieren organizar sus horarios escolares de forma rápida y sencilla. El programa te permite copiar directamente tus clases desde el sistema oficial de horarios de tu escuela, pegar el texto plano en la aplicación y generar automáticamente todas las combinaciones posibles de horarios según tus preferencias.

## ¿Para quién es?

Este proyecto está dirigido principalmente a estudiantes que necesitan visualizar, comparar y elegir el mejor horario posible de sus materias, sin perder tiempo en organizar manualmente sus clases.

## Características principales

- **Copiado directo**: Solo tienes que copiar la información de tus clases desde el sistema de tu escuela.
- **Pegado en texto plano**: Pega el texto en la aplicación, sin preocuparte por el formato.
- **Generador de horarios**: Schedule Wizard analizará las clases y generará todas las combinaciones posibles de horarios compatibles.
- **Filtros y preferencias**: Puedes seleccionar opciones según tus preferencias personales (por ejemplo, evitar huecos, elegir grupos específicos, etc).
- **Visualización amigable**: Los resultados se muestran de forma clara para que puedas comparar y elegir fácilmente.

## ¿Cómo usarlo?

1. **Copia tus clases** desde el sistema de horarios oficial de tu escuela.
2. **Pega el texto** en la sección indicada del programa.
3. **Personaliza tus preferencias** (opcional): puedes aplicar filtros para excluir horarios con huecos, seleccionar grupos, etc.
4. **Genera las opciones de horario**: el programa mostrará todas las combinaciones posibles de horarios compatibles.
5. **Revisa y elige** la opción que más te convenga.

## Requisitos

- Tener acceso al sistema de horarios de la escuela.
- Copiar la información de tus materias en texto plano.

## Instalación

1. Clona este repositorio:
   ```
   git clone https://github.com/CarlosFloresVaras/classtimematcherup
   ```
2. Instala las dependencias con npm:
   ```
   npm install
   ```
3. Inicia la aplicación:
   ```
   npm start
   ```
